# NLP Text Summarizer

A simple Natural Language Processing (NLP) web app built using Streamlit and Hugging Face Transformers.  
It takes a paragraph as input and generates a concise, meaningful summary instantly using the `facebook/bart-large-cnn` model.

---

## Features
- Accepts both short and long paragraphs
- Summarizes text instantly using a pre-trained transformer model
- Displays original and summarized word counts
- Option to download the summary as a text file
- Built with a simple and interactive Streamlit interface

---

##  Tech Stack
- Python 3.10+
- Streamlit
- Transformers (Hugging Face)
- PyTorch
